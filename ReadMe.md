# ООП, сезон второй

## Исходники

Клонировать репозиторий вместе с гугл тестами: `git clone --recurse-submodules https://github.com/ploschka/OOP-2.git`

## Сборка

Для сборки: `build.sh`.

## Запуск

После сборки для каждого исполняемого файла создадутся символьные ссылки в папке `exe`. Их запускать лучше из корневой директории репозитория, иначе могут повылазить ошибки о несуществующих csv файлах.
