# ООП, сезон второй
## Сборка
Для сборки сразу всего: просто запустить `build.sh`, если собирать всё не нужно, используйте эти флаги:
* -c для CalculatingMachine
* -p для Patterns
* -r для RegisterAndLogin
>Можно собрать сразу несколько, надо просто указать сразу несколько флагов

Модули собираются в любом случае

## Запуск
Запустить сразу всё не получится, поэтому просто запустить `run.sh` с одним из этих флагов:
* -c для CalculatingMachine
* -p для Patterns
* -r для RegisterAndLogin
